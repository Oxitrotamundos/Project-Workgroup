00 · Foundations /overview

Un sistema humanista para ingeniería de proyectos.

Lenguaje visual para una plataforma de gestión orientada a equipos de ingeniería: densa donde el desarrollador la necesita, ordenada para la lectura ejecutiva, y con calidez tipográfica para no sentir que se trabaja dentro de una hoja de cálculo.

Principio 01
Datos antes que adornos.
Cada elemento debe transportar información — números, estado, dueños, fechas — antes que ornamentos.
Principio 02
Densidad o respiro.
Tres densidades coherentes: vista de operación, vista por defecto y vista ejecutiva. Una sola variable cambia toda la jerarquía.
Principio 03
Tipografía con voz.
Sans humanista para texto, monoespaciada para datos y metadatos, serif itálica para titulares ejecutivos. Nunca todas a la vez.

06 · Foundations /motion

Movimiento contenido.

Toda transición usa la curva --ease. Tres duraciones.

fast · 120ms
Hover, foco, switch toggle, click feedback.
default · 180ms
Aparición de menús, tooltips, paneles colapsables.
slow · 320ms
Cambios de ruta, transiciones de vista de proyecto.
